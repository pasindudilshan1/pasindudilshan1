[![License](https://img.shields.io/github/license/pasindudilshan1/dcvpg?style=for-the-badge&color=03EF62)](https://github.com/pasindudilshan1/dcvpg/blob/main/LICENSE)

> **Catch schema drift, quality violations, and freshness SLA breaches before they reach production.**

```
🔍 Schema Drift Detection     → Catch column additions, drops & type changes automatically
✅ Data Quality Validation    → Assert nullability, ranges, regex patterns & referential rules
⏰ Freshness SLA Monitoring   → Alert when pipeline data goes stale beyond defined thresholds
🚦 Pipeline Gate Integration  → Block bad data at the source — before it poisons downstream
📋 Data Contract as Code      → Define and version your data contracts in YAML/JSON
```

**⭐ If dcvpg saves your pipeline, drop a star — it helps more data engineers find it!**
